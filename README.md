# Udacity Blockchain Capstone

The capstone will build upon the knowledge you have gained in the course in order to build a decentralized housing product. 

# Running the tests

## Setup

1. Run `npm install` inside main directory to get all the dependencies
2. Move to the eth-contracts directory by running `cd eth-contracts`
3. Start ganache, it should start on port 8545

## Running the tests

Run `truffle test` 

This should run all 3 tests inside the test folder under eth-contracts directory
